NS version 2.35 was used for this project.

Folder structure

Root folder contains 
1) TCL file (rtp.tcl)
2) The 1st version of the protocol implementation. (mac-mncprj.cc)
3) The header file for the same. (mac-mncprj.h)
4) The java analysis script file. (Script.java)

The folder v2 contains the 2nd version of the protocol implementation. (also named mac-mncprj.cc)

To vary the number of copies in the TCL file, the value of the variable repeatTimes must be changed.
